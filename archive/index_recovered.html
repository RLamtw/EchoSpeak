推薦方案 A：使用 Microsoft Edge 瀏覽器

Microsoft Edge 內建微軟著名的 Azure Neural 雲端語音。在 Edge 瀏覽器中開啟本網頁，可以直接在選擇語音角色清單中選取帶有「Online」的語音：

  • Microsoft Xiaoxiao Online (極自然的國語女聲)
  • Microsoft Yunxi Online (極自然的國語男聲)
  • 這些語音完全免費且無需任何 API 設定！

推薦方案 B：下載 macOS / iOS 的 Siri 高品質語音

Apple 提供極為自然流暢的 Siri 及提升型語音，可在系統中免費下載並授權給瀏覽器使用：

  1. 開啟 Mac 的 「系統設定」。
  2. 點選左側 「輔助使用」 ➔ 選擇 「語音內容」。
  3. 在「系統語音」下拉選單中選擇 「管理語音...」。
  4. 找到 「中文」 或 「英文」，下載帶有 「Siri」 或 「提升品質 (Premium)」 的語音（例如：美佳-提升品質）。
  5. 下載完成後，重新整理此網頁，即可在語音角色選單中看到並使用！

推薦方案 C：Chrome 瀏覽器線上語音

當 Chrome 保持網際網路連線時，會自動載入帶有 Google 標籤的線上語音角色。這些語音同樣比離線系統語音自然許多，可以直接在選單中選用。
